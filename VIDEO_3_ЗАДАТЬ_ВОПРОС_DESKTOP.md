# ВИДЕО 3 DESKTOP: Как задать вопрос AI-агенту на компьютере

**Цель:** Показать как ввести вопрос, получить ответ и продолжить диалог на компьютере

**Хронометраж:** 45-50 секунд

**Целевая аудитория:** Люди, далекие от компьютера

---

## ВАЖНО ПЕРЕД СЪЁМКОЙ

**Предусловие:** AI-агент уже открыт (см. Видео 2 Desktop)
**Пример вопроса:** Используйте простой, понятный вопрос (см. примеры ниже)

---

## УНИВЕРСАЛЬНАЯ ВЕРСИЯ (Windows и Mac - без различий)

### Кадр 1: Заставка (0:00-0:03)
**Визуал:** Текст на экране
```
КАК ЗАДАТЬ ВОПРОС
AI-АГЕНТУ
Компьютер
```

**Озвучка:** "Покажу, как общаться с AI-помощником на компьютере."

---

### Кадр 2: Окно агента открыто (0:03-0:08)
**Визуал:**
- Показать открытое окно AI-агента
- Видно поле для ввода текста (обычно внизу экрана)
- Стрелка указывает на поле ввода
- Подпись: "Поле для вопроса"
- Курсор мыши находится рядом с полем

**Озвучка:** "Агент уже открыт. Внизу вы видите поле для вопроса."

---

### Кадр 3: Нажать на поле ввода (0:08-0:12)
**Визуал:**
- Крупный план: поле ввода
- Стрелка указывает на поле
- Курсор мыши движется к полю ввода
- Клик по полю (показать анимацию клика)
- Поле становится активным (мигающий курсор внутри)
- Возможно изменение цвета рамки поля (синяя подсветка)

**Озвучка:** "Нажмите на это поле. Оно станет активным."

**Действие:** Клик мышью по полю ввода

---

### Кадр 4: Ввод вопроса (0:12-0:25)
**Визуал:**
- Поле ввода активно, курсор мигает
- Начать печатать вопрос (медленно, по буквам)
- **Пример вопроса:** "У меня болят суставы, что посоветуете?"
- Показать как текст появляется в поле по мере печати
- ВАЖНО: Печатать достаточно медленно, чтобы зритель успел прочитать
- Можно показать руки на клавиатуре (опционально)

**Озвучка:** "Введите ваш вопрос с клавиатуры. Например: У меня болят суставы, что посоветуете? Пишите простым языком, как будто разговариваете."

**Действие:** Набор текста с клавиатуры

---

### Кадр 5: Отправить вопрос (0:25-0:30)
**Визуал:**
- Весь вопрос введен в поле
- Стрелка указывает на кнопку отправки (обычно значок стрелки или кнопка "Отправить")
- Курсор наводится на кнопку отправки
- Кнопка может подсветиться (hover-эффект)
- Клик по кнопке (показать анимацию)
- Вопрос отправляется (поле очищается или появляется индикатор загрузки)

**Озвучка:** "Теперь нажмите кнопку Отправить. Вот эту стрелочку."

**Действие:** Клик по кнопке отправки

**АЛЬТЕРНАТИВА:** Можно нажать Enter на клавиатуре вместо кнопки

---

### Кадр 6: Получение ответа (0:30-0:40)
**Визуал:**
- Показать как появляется ответ агента
- Возможна анимация печати (текст появляется постепенно) или сразу весь текст
- Крупный план: текст ответа агента
- Прокрутить ответ вниз с помощью мыши (колесико или скроллбар), чтобы показать, что он может быть длинным
- Показать структуру ответа: рекомендация продуктов, дозировки, советы

**Озвучка:** "Агент отвечает. Он посоветует продукты и расскажет как их принимать. Ответ может быть длинным, прокрутите вниз колесиком мыши."

---

### Кадр 7: Задать уточняющий вопрос (0:40-0:48)
**Визуал:**
- Прокрутить страницу вниз до поля ввода
- Курсор мыши снова кликает на поле ввода
- Ввести короткий уточняющий вопрос: "А сколько стоит?"
- Нажать отправить (или Enter)
- Показать новый ответ агента (кратко)

**Озвучка:** "Можете задать еще вопрос. Например, про цену. Агент ответит."

**Действие:**
1. Клик по полю ввода
2. Ввод уточняющего вопроса
3. Отправка (клик или Enter)

---

### Кадр 8: Итог (0:48-0:50)
**Визуал:**
- Показать весь диалог: ваш вопрос → ответ агента → уточнение → ответ
- Текст на экране:
```
✅ ГОТОВО!
Вы можете задавать сколько угодно вопросов
```

**Озвучка:** "Вот и всё! Можете задавать столько вопросов, сколько нужно. Агент всегда ответит."

---

## ПРИМЕРЫ ВОПРОСОВ ДЛЯ СЪЁМКИ

Выберите один из этих вопросов для демонстрации:

**Простые:**
1. "У меня болят суставы, что посоветуете?"
2. "Хочу укрепить иммунитет"
3. "Плохо сплю по ночам"
4. "Нужны витамины для кожи"

**С уточнением:**
5. "Есть что-то для энергии?"

**Уточняющие вопросы (для второго этапа):**
- "А сколько стоит?"
- "Как долго принимать?"
- "Есть ли противопоказания?"
- "Можно ли детям?"

---

## КЛЮЧЕВЫЕ МОМЕНТЫ ДЛЯ СЪЁМКИ

1. **Активное поле:** Показать как поле ввода меняется при клике (подсветка, мигающий курсор)
2. **Медленная печать:** Набирать текст медленно, с паузами между словами
3. **Крупный план:** Показать поле ввода крупно, чтобы было видно текст
4. **Прокрутка:** Обязательно показать прокрутку длинного ответа колесиком мыши
5. **Демонстрация диалога:** Важно показать, что можно задать несколько вопросов подряд

---

## РАЗЛИЧИЯ Windows vs Mac

| Элемент | Windows | Mac |
|---------|---------|-----|
| Клавиатура | Windows клавиатура | Mac клавиатура |
| Отправка | Enter или кнопка | Return (Enter) или кнопка |
| Прокрутка | Колесико мыши или скроллбар | Колесико мыши, трекпад или скроллбар |
| Все остальное | ОДИНАКОВО | ОДИНАКОВО |

**Вывод:** Можно снять одно универсальное видео. Различия только в виде клавиатуры, но действия идентичны.

## ЧАСТЫЕ ОШИБКИ (предупредить в описании)

**Если агент не отвечает:**
- Проверьте интернет-соединение
- Подождите 10-15 секунд (агент думает)
- Попробуйте обновить страницу (F5 на Windows, Cmd+R на Mac)

**Как задавать вопросы:**
- Пишите простым языком
- Можно задавать несколько вопросов подряд
- Не нужно быть вежливым ("пожалуйста" не обязательно)
- Агент понимает множество языков, включая русский, казахский, английский

**Отправка сообщения:**
- Можно нажать кнопку "Отправить"
- Или просто нажать Enter на клавиатуре

## СОВЕТЫ ДЛЯ МОНТАЖА

**Визуальные элементы:**
- Яркие стрелки для указания на поле ввода и кнопку отправки
- Увеличение курсора мыши для лучшей видимости
- Анимация клика (круг при нажатии)
- Выделение активного поля (можно обвести рамкой)
- Показать руки на клавиатуре при печати (опционально, но помогает)

**Текстовые подсказки:**
- "Нажмите на поле"
- "Введите вопрос"
- "Нажмите Отправить"
- "Прокрутите вниз"
