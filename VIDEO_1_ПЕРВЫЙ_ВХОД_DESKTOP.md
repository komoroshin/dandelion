# ВИДЕО 1 DESKTOP: Первый вход в личный кабинет с компьютера

**Цель:** Показать как впервые зайти в личный кабинет Dandelion с компьютера

**Хронометраж:** 40-45 секунд

**Целевая аудитория:** Люди, далекие от компьютера

---

## ВАЖНО ПЕРЕД СЪЁМКОЙ

**Браузеры:** Используйте Chrome или Safari (НЕ Yandex, НЕ Firefox)
**VPN:** НЕ НУЖЕН! Агент работает без VPN

---

## УНИВЕРСАЛЬНАЯ ВЕРСИЯ (Windows и Mac)

### Кадр 1: Заставка (0:00-0:03)
**Визуал:** Текст на экране
```
ПЕРВЫЙ ВХОД В ЛИЧНЫЙ КАБИНЕТ DANDELION
Инструкция для компьютера
```

**Озвучка:** "Привет! Сейчас покажу, как зайти в личный кабинет Dandelion с компьютера."

---

### Кадр 2: Рабочий стол и браузер (0:03-0:10)
**Визуал:**
- Показать рабочий стол компьютера
- Стрелка указывает на иконку браузера Chrome (разноцветный круг) или Safari (синий компас)
- Курсор мыши движется к иконке
- Двойной клик по иконке браузера
- Браузер открывается

**Озвучка:** "Найдите на рабочем столе браузер Chrome или Safari. Откройте его двойным щелчком мыши."

**Действие:** Двойной клик по иконке браузера

**РАЗЛИЧИЯ:**
- **Windows:** Обычно Chrome (разноцветный круг)
- **Mac:** Может быть Safari (синий компас) или Chrome

---

### Кадр 3: Адресная строка (0:10-0:20)
**Визуал:**
- Браузер открыт, показана пустая страница или главная
- Крупный план: адресная строка вверху окна
- Стрелка указывает на адресную строку
- Курсор мыши щелкает по адресной строке
- Адресная строка становится активной (мигающий курсор)
- Вводится адрес: **v3.dandelion.club**
- Подпись на экране: "Адрес вам дал спонсор"

**Озвучка:** "Нажмите на строку вверху окна. Введите адрес сайта. Его вам отправил ваш спонсор. Затем нажмите Enter."

**Действие:**
1. Клик по адресной строке
2. Ввод URL с клавиатуры
3. Нажатие Enter

---

### Кадр 4: Страница входа (0:20-0:35)
**Визуал:**
- Открылась страница входа в личный кабинет
- Показать форму с двумя полями и кнопкой
- Стрелка указывает на первое поле: "Телефон" или "Email"
- Курсор щелкает в первое поле
- Ввод телефона/email с клавиатуры
- Стрелка переходит ко второму полю: "Пароль"
- Курсор щелкает во второе поле
- Ввод пароля (символы заменяются точками или звездочками)
- Стрелка указывает на кнопку "Войти"
- Курсор наводится на кнопку, она меняет цвет (hover-эффект)
- Клик по кнопке "Войти"

**Озвучка:** "Введите ваш номер телефона или электронную почту. Нажмите в следующее поле. Введите пароль. Эти данные вам дал спонсор. Нажмите кнопку Войти."

**Действие:**
1. Клик в поле "Телефон/Email"
2. Ввод данных с клавиатуры
3. Клик в поле "Пароль" (или нажатие Tab)
4. Ввод пароля
5. Клик по кнопке "Войти"

---

### Кадр 5: Успех + Предупреждение (0:35-0:45)
**Визуал:**
- Страница обновилась
- Личный кабинет открылся (главная страница)
- Показать основные элементы интерфейса
- Текст на экране:
```
✅ ГОТОВО!
Вы вошли в личный кабинет

⚠️ Используйте ТОЛЬКО Chrome или Safari
❌ Без VPN
```

**Озвучка:** "Готово! Вы вошли в личный кабинет. Важно запомните: всегда используйте Chrome или Safari. VPN не нужен."

## КЛЮЧЕВЫЕ РАЗЛИЧИЯ Windows vs Mac

| Элемент | Windows | Mac |
|---------|---------|-----|
| Браузер по умолчанию | Chrome (обычно) | Safari или Chrome |
| Иконка Chrome | Разноцветный круг | Разноцветный круг |
| Иконка Safari | Нет | Синий компас |
| Клавиша Enter | Enter | Return (или Enter) |
| Переход между полями | Tab | Tab |

**Вывод:** Различия минимальны. Можно снять одно универсальное видео, упомянув оба браузера.

## СОВЕТЫ ДЛЯ СЪЁМКИ

1. **Крупные планы:** Показывать курсор мыши, который движется и нажимает
2. **Медленный темп:** Делать паузы после каждого действия
3. **Выделение элементов:** Обводить ярким цветом адресную строку, поля ввода, кнопки
4. **Клавиатура:** Можно показать крупным планом как печатаются символы
5. **Активные элементы:** Показать изменение цвета кнопок при наведении (hover)

## ЧАСТЫЕ ОШИБКИ (предупредить)

**Если страница не открывается:**
- Проверьте правильность адреса (буквы, точки)
- Убедитесь что есть интернет
- Попробуйте другой браузер (Chrome или Safari)

**Если не получается войти:**
- Проверьте правильность телефона/email (нет лишних пробелов)
- Проверьте пароль (большие/маленькие буквы имеют значение)
- Свяжитесь со спонсором для уточнения данных

**Если требует VPN:**
- VPN НЕ НУЖЕН! Отключите его если включен
- Если не работает без VPN - напишите спонсору
